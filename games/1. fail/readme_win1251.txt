Для запуска игры необходим интерпретатор INSTEAD не ниже версии 1.8.
Адрес проекта: http://instead.syscall.ru/

Для установки поместите распакованную папку с игрой в директорию "games", что появится при установке. 
Для Windows это: <куда установили>/INSTEAD/games
Для Linux: ~$.instead/games
